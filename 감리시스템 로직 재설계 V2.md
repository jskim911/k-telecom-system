1\. 공정관리 메뉴 설계 로직

&#x20; - 화면설계는 게시판 형태의 화면, 탭메뉴, 기능 버튼으로 구성한다. 기능버튼은 1)**작업발행**, 2)**일보발행**

&#x20; - 게시판 탭메뉴에는 작업지시서, 작업일보, 감리일보 3개로 구성한다.

&#x20;  - 작업발행 버튼을 클릭하면 작업지시서 탭메뉴에 제목이 리스트 형태로 작업지시서가 표시된다.

&#x20;  - **작업버튼 클릭시** 화면설계 입력모달창에 데이터를 입력후 생성하면 작업지시서가 발행된다.

&#x20; - 작업지시서 제목은 "**작업날짜+작업위치+작업공정+일련번호**" \*일련번호\* 부여는 자동으로 부여한다.

&#x20; - 작업지시 리스트 제목 좌측에 **체크박스**, 우측에 작업공정률 관리할 수 있는 **상태/진도율** 기능을 넣는다.

&#x20;  - 체크박스 선택후 **일보발행** 버튼을 클릭하면 **해당 작업지시서와 연동된** 해당 탭메뉴에 **작업일보**와 **감리일보**에 리스트가 자동생성한다.

&#x20;  - 작업일보 탭의 작업일보 리스트를 클릭하면 작업일보 양식을 구글 드라이브에서 불러와서 작성할 수 있게 한다.

&#x20;  - 감리일보 탭의 감리일보 리스트를 클릭하면 감리일보 양식을 구글 드라이브에서 불러와서 작성할 수 있게 한다.

&#x20;  - 해당 일보를 작성하여 저장하면 구글드라이브의 "해당 공사명/출력문서/하위폴더"(작업일보, 감리일보)에 저장하여 해당 리스트와 연동되도록 한다.



2\. 양식관리 메뉴 신설 : \* **감리일보, 품질/검측, 안전관리 메뉴 삭제** -> 공정관리 메뉴와 양식관리로 이전

&#x20; - 카드형태의 기능버튼으로 화면을 구성한다. 2x2배열로 구성한다.

&#x20;  - 카드 메뉴 종류로는 **품질/검측**, **안전관리**, **준공관리**, **환경관리** 버튼을 구성한다.

&#x20;  - 준공관리, 환경관리 버튼은 공정메뉴/작업일보 탭메뉴의 작업일보와 연동하여 해당 리스트를 체크하면 해당 리스트와 연동되어 양식파일이 관리되어야 한다.

&#x20;  - 품질/검측, 안전관리 버튼은 공정관리/감리일보 탭메뉴의 감리일보와 연동되어 해당 리스트를 체크하면 해당 리스트와 연동되어 양식파일이 관리되어야 한다. 

&#x20;  - 해당 리스트 선택하고 완료하면 구글 드라이브 각종 양식 템플릿에서 해당 양식파일을 선택하면 양식 템플릿 파일을 불러와서 양식에 데이터 입력하고 저장하면 해당 출력문서 폴더에 저장되어야 한다. (공통)

&#x20;  - 즉, 카드 메뉴를 클릭하면 리스트 선택, 구글 드라이브의 양식 템플릿 파일을 선택하고  완료누르면, 선택한 리스트와 연동되어 작성 하고 저장한 양식파일 관리되는 것임.
