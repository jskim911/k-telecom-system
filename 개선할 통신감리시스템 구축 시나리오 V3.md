개선 할 통신감리시스템 구축 시나리오 



1\. 전체 개요

&nbsp; - 로그인 기능을 만든다. 로그인 관리를 위한 관리자 기능 필요 : 회원 가입 승인, 삭제 

&nbsp; - 프로젝트(공사)별로 아래 기능이 관리되도록 해야 한다.

&nbsp; - 공정관리 정보 생성으로 감리일보, 안전관리, 품질관리, 환경관리 등 시작된다.

&nbsp; - 분야별 작성되는 서류는 기본정보와 추가정보로 구분된다.

&nbsp; - 기본정보는 공종별 가중치(공종별 수량대비), 전체 가중치(노무비 금액 대비), 작업자



2\. 공정관리 메뉴 : 기본정보

&nbsp; - 공종항목 생성/추가 : **설계서 노무비 항목을 기준**으로 생성한다. (광케이블 설치, LAN케이블 설치 등)

&nbsp; - 생성 버튼 : 입력모달창(리스트 형태 항목추가) -> 완료버튼-> 공종항목 리스트 생성

&nbsp; - 공종항목 리스트의 추가/수정버튼 : "작업일시, 작업장소, 작업자, 작업량, 공정율" 입력 모달창 생성

&nbsp; - 생성될 작업리스트 항목 "작업일시+공종항목" 제목으로 리스트로 생성 

&nbsp; - 생성된 작업내용 리스트 항목을 클릭하면 입력된 세부내용이 보여준다. 수정 버튼 필요

&nbsp; - 또한, 작업리스트 항목에 서류생성 버튼 필요 : 현장 관리되는 각종 일보, 작업내용 등 생성할 리스트

&nbsp; **- 필요한 리스트를 선택하고 완료 -> 감리일보, 안전일보, 품질서류 등(사전에 정의)  생성 기능**

&nbsp; - 해당 메뉴(김리일보, 안전관리 등)에 "작업일시+작업리스트" 제목으로 리스트 생성 

&nbsp; - 완료 버튼 : 작업일시 기준으로 생성된 세부 작업내용이 리스트로 보여준다

&nbsp;

3\. 감리일보 메뉴 :  추가정보

&nbsp; - 사전에 사용할 표준화된 서류양식 등록 : 데이터 입력, 출력내용 사전에 정의한다.

&nbsp; - 표준 양식 업로드 -> 입력 모달창 자동 생성, 출력 양식은 업로드된 양식을 사용한다.

&nbsp; - 공정관리에서 생성한 "날짜+작업리스트" 리스트 클릭-> 입력 모달창 -> 작업내용 입력



4\. 안전관리 메뉴 :  추가정보

&nbsp; - 사전에 사용할 표준화된 서류양식 등록 : 데이터 입력, 출력내용 사전에 정의한다.

&nbsp; - 표준 양식 업로드 -> 입력 모달창 자동 생성, 출력 양식은 업로드된 양식을 사용한다.

&nbsp; - 공정관리에서 생성한 "날짜+작업리스트" 리스트 클릭-> 입력 모달창 -> 작업내용 입력



5\. 품질관리 메뉴 :  추가정보

&nbsp; - 사전에 사용할 표준화된 서류양식 등록 : 데이터 입력, 출력내용 사전에 정의한다.

&nbsp; - 표준 양식 업로드 -> 입력 모달창 자동 생성, 출력 양식은 업로드된 양식을 사용한다.
